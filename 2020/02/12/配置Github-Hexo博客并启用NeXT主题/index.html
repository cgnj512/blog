然后点击New Repository（https://github.com/new）新建一个仓库
假设你的Github用户名是aUser_name
则Repository name填 aUser_name.github.io
Description指对博客的描述，随便写，或者不写
下面选Public，博客当然要公开
再下面啥都不要点，方框不要打钩，选择框全部None，点击Create，完成。
